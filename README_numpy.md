# 🔢 NumPy Project Notebook

This project contains a Jupyter Notebook demonstrating the usage of the
**NumPy** library for numerical computing in Python.

------------------------------------------------------------------------

## 📌 Features

-   Array creation and manipulation\
-   Mathematical operations on arrays\
-   Indexing and slicing\
-   Performance advantages over Python lists

------------------------------------------------------------------------

## 🧾 About the Notebook

The notebook covers: - Importing NumPy\
- Creating arrays (1D, 2D, multi-dimensional)\
- Array operations (addition, multiplication, etc.)\
- Statistical functions (mean, median, std)\
- Reshaping and broadcasting

------------------------------------------------------------------------

## 🧑‍💻 Requirements

Ensure the following are installed:

-   Python 3.x\
-   Jupyter Notebook / JupyterLab\
-   numpy

Install dependencies:

``` bash
pip install numpy notebook
```

------------------------------------------------------------------------

## ▶️ How to Run

1.  Download or clone the project\
2.  Navigate to the folder\
3.  Launch Jupyter Notebook:

``` bash
jupyter notebook
```

4.  Open the `.ipynb` file\
5.  Run cells step by step
